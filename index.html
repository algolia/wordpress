Override the native WordPress search to benefit from all the Algolia features like:

  • Typo tolerance,
  • Remove stop words,
  • Ranking based on custom metrics,
  • ...

Automatic synchronization

Choose the post types & taxonomies you want to index in Algolia and the plugin will make sure it stays in sync.

Analytics

A lot of WordPress hooks and filters are introduced by the plugin so that developer can easily:

  • Customize the data pushed to Algolia,
  • Change the ranking according to business metrics,
  • And more...
